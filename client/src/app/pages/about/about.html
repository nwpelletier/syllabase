About

What this app will do:

  • Store information about musical pieces, composers, and collections
  • Allow searching and filtering of pieces by various criteria
  • Provide a user-friendly interface for managing the database
  • Support for themes (light/dark mode)
  • Responsive design for use on various devices

Technologies used:

  • Frontend: Angular, TypeScript, HTML, CSS
  • Backend: Node.js, NestJS, TypeScript
  • Database: PostgreSQL, TypeORM
  • Version Control: Git, GitHub
